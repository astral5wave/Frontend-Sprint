# Frontend Sprint

A collection of 24 mini frontend projects built for hands-on practice and to strengthen skills in the React ecosystem. Each project demonstrates a unique UI/UX feature or interaction. All projects are available and can be tried live at [frontendsprint.netlify.app](https://frontendsprint.netlify.app/). Many more projects will be added in the future.

## Projects

- **Accordion** – Expand/collapse sections.  
- **Audio Player** – Play audio with controls.  
- **Button Ripple Effect** – Interactive button animation.  
- **Clock** – Live digital clock.  
- **Currency Converter** – Convert currencies dynamically.  
- **Drag and Drop** – Drag items between containers.  
- **Expense Tracker** – Track and manage expenses.  
- **Folder Tree View** – Nested folder structure UI.   
- **Image Slider** – Carousel with navigation controls.  
- **Infinite Scroll** – Load content as you scroll.  
- **Modal with Outside Click** – Close modal when clicking outside.  
- **Nested Comments** – Threaded comment system.  
- **Pagination** – Split content across multiple pages.  
- **Quiz App** – Interactive quiz with scoring.  
- **Random Color** – Generate and display random colors.  
- **Scroll Indicator** – Show page scroll progress.  
- **Scroll to Section** – Smooth scrolling to sections.  
- **Search Autocomplete** – Suggest results as you type.  
- **Star Rating** – Rate items with stars.  
- **Step Progress Bar** – Multi-step progress visualization.  
- **Tic Tac Toe** – Classic game implementation.  
- **Toggle Theme** – Switch between light and dark modes.  
- **Tooltip** – Show contextual information on hover.  
- **Weather App** – Display weather information dynamically.  

## Purpose

This collection helped me practice React fundamentals, component design, state management, and UI interactions in real projects. More projects will be added as the collection grows.
